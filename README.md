# Bank Management System (Python + Tkinter + SQLite)

A simple and secure Bank Management System built using Python, Tkinter, and SQLite.  
This project allows users to create accounts, log in securely, and perform basic banking operations such as deposit, withdraw, and balance check.  
It is designed for beginners to understand GUI development and database handling using Python.
